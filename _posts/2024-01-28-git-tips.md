## 1. よく使うコマンド

```sh
git add -n .
git log --graph
git diff --name-only HEAD HEAD~2
git diff ID1 ID2 filename
git diff ID1 ID2 filename
git difftool ID1 ID2
git show ID

git reflog
git rebase
```

## 2. リモートリポジトリ作成

```sh
git init --bare --shared
git remote add origin url
git clone url
```
